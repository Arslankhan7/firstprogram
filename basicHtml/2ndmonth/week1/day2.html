CSS

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML (Hypertext Markup Language). CSS separates the structure of a document from its style, allowing web designers and developers to control the layout and appearance of web pages.

C++

C++ is a general-purpose programming language that was
developed as an extension of the C programming language.
It was designed by Bjarne Stroustrup and first appeared in 1983.
C++ adds object-oriented programming features to C,
providing developers with additional tools for
creating complex and modular software.

JAVA

Java is a versatile, object-oriented programming language developed by Sun Microsystems (now owned by Oracle Corporation) in the mid-1990s. It was created by James Gosling and his team with the goal of providing a platform-independent language that could be used for developing a wide range of applications. Java quickly gained popularity due to its "Write Once, Run Anywhere" (WORA) principle, which means that Java code can run on any device or platform that has a Java Virtual Machine (JVM).
